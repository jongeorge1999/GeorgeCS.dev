Graphics Engine

This is my personal engine that I am developing from the ground up in C++/OpenGL. Used for general rendering and graphics programming.

Features

  • OBJ loading/processing
  • UI with Scene/Hierarchy/Inspector
  • Lighting/Shadows
  • Postprocessing options (MSAA, filter, blur, etc.)
  • Lightweight Entity Component System
  • Inspector controlled transforms, widgets
  • Instanced Rendering
  • Normal/Parallax Mapping
  • Skyboxes
  • Reflection cubemaps
  • Transparency
  • Normal vector visualization
