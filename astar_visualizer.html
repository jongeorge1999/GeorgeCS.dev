A* Pathfinding Visualizer

My Interactive A* pathfinding visualizer where the user can draw barriers and have the closest path be recalculated in real time.

View Code on GitHub: https://github.com/jongeorge1999/AstarVisualizer

Controls

  • Left-Click: Places Barrier at mouse Position
  • Right-Click: Places Goal
  • Middle-Click: Places Pathfinder
  • P: Start Pathfinding
  • O: Pause Pathfinding
  • R: Reset Canvas
